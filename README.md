## ✨ 核心功能

本工具集成了 10 种实用的 PDF 处理工作流面板：

1. **📝 Word 与 PDF 双向转换**
   - 拖入 `.pdf` 文件自动转换输出为 `.docx` 格式（需环境中存在 `pdf2docx`）。
   - 拖入 `.docx` 文件自动转换输出为 `.pdf` 格式（需环境中存在 `docx2pdf` 且本地已安装 Office）。
2. **📄 PDF 转图片**
   - 支持将 PDF 各页面渲染输出为 PNG/JPG。
   - 提供 72、150、300、600 DPI 档位供选择。
3. **🖼️ 图片转 PDF**
   - 导入主流格式（JPG, JPEG, PNG, WEBP, BMP）并转换为 PDF。
   - 支持“合并为一个 PDF”或“分别转为独立 PDF”模式，并可通过右键菜单灵活调整合并时的图片顺序。
4. **📑 PDF 多文件合并**
   - 支持批量导入多个 PDF 文件，通过交互表格自由进行“上移/下移/移除”排序，随后按序合并输出。
5. **✂️ PDF 页码提取/删除**
   - 支持输入指定页码段（例如：`1-3, 5`），一键选择“保留选中页码”或“删除选中页码”进行分割。
6. **🗜️ PDF 引擎压缩**
   - 采用 PyMuPDF 底层的垃圾回收和数据流压缩机制，在尽可能不改变视觉清晰度的前提下缩减文件体积。
7. **🔍 PDF 深度内容提取**
   - 提取 PDF 中的纯文本内容（`.txt`）、提取所有内嵌图片（`.png`）、提取表格数据（`.json`）或提取矢量绘图路径（`.json`）。
8. **🛡️ PDF 安全与防泄漏**
   - 快速进行 AES-256 标准的 PDF 密码加密保护。
   - 输入原密码移除密码限制。
   - 物理擦除（打黑打码）：对文档中的敏感文字进行像素级底层物理覆盖擦除。
9. **📐 PDF 页面排版与调整**
   - 提供顺/逆时针页面旋转、自定义坐标裁剪、添加图片水印、尾部追加空白页或写入特定文本。
   - 进阶拼版：支持将单页拆分为四页（海报化）及将四页合一拼接为单页（拼版打印）。
10. **📎 PDF 附件与文档嵌入**
    - 支持在页面首页左上角以“图钉图标”附加外部文件。
    - 支持作为隐藏附件，完全嵌入到 PDF 文档结构内部。

## 🛠️ 技术特点

- **Obsidian 极简美学**：全局基于深蓝色调（`#0f172a` 与 `#090d16`）定制 QSS 样式表，包含细致的悬停态动效与定制化滚动条设计。
- **动态交互引导**：界面输入控件与下拉模式智能绑定。切换至无需填写参数的模式时，文本输入框会自动转换为空白提示并禁用，避免误输入。
- **异步多线程**：所有转换和提取任务通过后台 `QThread` 执行，保证 UI 界面在批量任务运行时不会出现卡顿或无响应，支持实时的单文件进度更新。
- **拖拽与路径遍历**：Drop 拖拽区域不仅支持直接拖入单个或多个目标文件，在拖入文件夹时还会自动检索子目录下的所有支持格式。

## 🚀 快速开始

### 1. 克隆或下载本仓库
```bash
git clone https://github.com/Qwejay/QpdfTool.git
cd QpdfTool
```

### 2. 安装环境依赖
请确保系统已安装 **Python 3.8+**。在终端中运行以下命令安装基础运行依赖：

```bash
pip install PySide6 PyMuPDF
```

若需使用 **Word 互转** 功能，请根据需要补充安装以下可选依赖库：

```bash
# 用于 PDF 转换为 Word
pip install pdf2docx

# 用于 Word 转换为 PDF (注: docx2pdf 的运行需依赖系统本地已安装 Microsoft Word 软件)
pip install docx2pdf
```

### 3. 运行主程序
```bash
python main.py
```

* **Windows**：优先使用 `Segoe UI`
